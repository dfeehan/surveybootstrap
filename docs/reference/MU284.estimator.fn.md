# MU284.estimator.fn

Produce estimates from a simulated sample survey of the
[MU284](https://dfeehan.github.io/surveybootstrap/reference/MU284.md)
population. Used in package tests and examples.

## Usage

``` r
MU284.estimator.fn(survey.data, weights)
```

## Arguments

- survey.data:

  the survey dataset

- weights:

  a vector with the survey weights

## Value

a data.frame with one row and two columns:

- `TS82.hat` - the estimated total of `S82`

- `R.RMT85.P85.hat` - the estimated ratio of `RMT85` / `P85`
