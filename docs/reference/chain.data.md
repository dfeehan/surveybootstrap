Get a dataset from a chain
==========================

<div class="ref-description section level2">

Take the data for each member of the given chain and assemble it
together in a dataset.

</div>

<div class="section level2">

Usage
-----

<div class="sourceCode">

``` r
chain.data(chain)
```

</div>

</div>

<div class="section level2">

Arguments
---------

-   chain:

    The chain to build a dataset from

</div>

<div class="section level2">

Value
-----

A dataset comprised of all of the chain's members' data put together.
The order of the rows in the dataset is not specified.

</div>
