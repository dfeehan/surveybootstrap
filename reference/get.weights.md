# Get the weights column from a dataframe

This is the same as
[`get.var()`](http://dennisfeehan.org/surveybootstrap/reference/get.var.md)
with the default value set to 1 instead of `NA`

## Usage

``` r
get.weights(...)
```

## Arguments

- ...:

  (this is a function curried from
  [`get.var()`](http://dennisfeehan.org/surveybootstrap/reference/get.var.md))
