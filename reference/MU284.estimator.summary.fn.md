# MU284.estimator.summary.fn

Summarize results from
[`MU284.estimator.fn()`](http://dennisfeehan.org/surveybootstrap/reference/MU284.estimator.fn.md)
applied to many surveys. (This is a dummy function, used for tests)

## Usage

``` r
MU284.estimator.summary.fn(res)
```

## Arguments

- res:

  a dataframe whose rows are the results of calling
  [`MU284.estimator.fn()`](http://dennisfeehan.org/surveybootstrap/reference/MU284.estimator.fn.md)

## Value

the same dataframe
